# 📊 Vendor Sales & Inventory Analysis

## 📌 Project Overview

This project focuses on analyzing vendor performance, sales trends, and inventory efficiency using multiple datasets. The goal is to transform raw data into meaningful insights that support business decision-making.

---

## 🧠 Objectives

* Analyze vendor performance and profitability
* Identify top-performing and low-performing vendors and brands
* Detect unsold inventory and inefficiencies
* Understand sales trends and pricing impact
* Support data-driven decisions using visualization and statistical analysis

---

## 🗂️ Dataset

The project uses multiple datasets:

* Purchases
* Sales
* Vendor Invoice
* Purchase Prices

These datasets were cleaned, transformed, and merged to create a final analytical dataset.

---

## 🛠️ Tools & Technologies

* **Python (Pandas, NumPy)** → Data cleaning & transformation
* **SQL Server** → Data storage & querying
* **Power BI** → Dashboard & visualization
* **Jupyter Notebook** → Analysis & hypothesis testing

---

## 🔄 Data Processing Steps

1. Handled missing values and inconsistencies
2. Standardized column names across datasets
3. Merged multiple tables using appropriate joins
4. Aggregated data to calculate key metrics
5. Created new features such as **Gross Profit**

---

## 📊 Key Metrics

* Total Sales
* Total Purchase Cost
* Freight Cost
* Gross Profit
* Sales Quantity
* Profit Margin

---

## 📈 Analysis Performed

* Exploratory Data Analysis (EDA)
* Trend Analysis
* Vendor & Brand Performance Analysis
* Hypothesis Testing

---

## 📌 Key Insights

* A small number of vendors contribute the majority of sales and profit
* Several products have zero sales despite purchases, indicating unsold inventory
* Higher-priced products do not always generate higher sales
* Profitability varies significantly across brands

---

## 💡 Recommendations

* Focus on high-performing vendors to maximize profit
* Reduce procurement of low-selling products
* Improve inventory planning to minimize unsold stock
* Optimize pricing strategies based on demand patterns

---

## 📊 Dashboard

An interactive Power BI dashboard was created to visualize:

* Sales trends
* Vendor performance
* Brand contribution
* Profitability and inventory insights

---

## 🎯 Conclusion

This project demonstrates an end-to-end data analysis workflow, from data cleaning and transformation to visualization and business insights generation.
